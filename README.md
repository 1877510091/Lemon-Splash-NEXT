# 柠檬单词 NEXT 
柠檬单词 —— 一款清新治愈、纯净高效的沉浸式背单词 App。拒绝枯燥，让学习像喝柠檬水一样清爽。
🌟 产品理念 背单词不应该是一件痛苦枯燥的苦差事。Lemon Splash (柠檬单词) 采用独创的“柠檬海”透亮磨砂 UI 设计，将学习过程视觉化为清新的收集体验。我们致力于在繁重的备考压力下，为你提供一方纯净、高效、零干扰的学习净土。

✨ 核心功能

🌊 极致清新的视觉体验 告别传统刻板的背词界面，采用全屏动态背景与高斯模糊（Glassmorphism）卡片设计。每一次点击都伴随着清爽的交互反馈，有效缓解视觉疲劳，让注意力更聚焦。

📚 覆盖全阶段权威词库 内置 CET-4、CET-6、考研英语、托福 (TOEFL)、雅思 (IELTS) 五大核心词库。无论你是大学生备考，还是计划出国深造，都能在这里找到精准的学习内容。支持一键秒速切换词库，学习计划随心定。

⚡ 科学分组，高效记忆 采用 “20词/组” 的科学切片学习法，将庞大的词汇量化整为零，大幅降低记忆心理负担。

智能循环：每组学完后支持“立即复习”或“进入下一组”。

断点续传：自动保存学习进度，精准记录你学到了哪一组，随时退出，随时继续，绝不丢失进度。

📈 可视化数据统计 你的每一份努力都值得被记录。

每日收集：实时显示今日学习

月度趋势：强大的交互式折线图，清晰展示每月、每日的学习波动，助你复盘学习状态，养成坚持的好习惯。

🛡️ 纯净离线，隐私安全

完全离线：基于 SQLite 本地数据库，无网络环境也能流畅背词（图书馆/地铁党的福音）。

零干扰：无广告、无社交、无推送，还原最纯粹的学习环境。

🚀 适用人群

正在备战四六级、考研的在校大学生。

准备雅思、托福考试的留学党。

希望提升英语词汇量，且追求 App 颜值与体验的终身学习者。

【技术亮点】
架构：基于 Google Flutter 框架开发，极致流畅的跨平台体验。

存储：采用 SQLite (sqflite) 处理海量词库，结合 SharedPreferences 管理轻量级进度，实现秒级启动与数据持久化。

图表：集成 FL Chart 引擎，通过复杂的计算逻辑渲染高性能交互式折线图。

算法：内置异步分批导入算法，支持在后台线程解析大型 JSON 数据，确保 UI 线程永不卡顿。
